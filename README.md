# HTML5 Chatroom

For 資策會 前端工程師就業養成班MFEE13 - HTML5 JS API

## Environment

- nodejs
- express
  - pug


## Start website for dev

- for mac or linux
```bash
DEBUG=html5-chatroom:* npm run devstart
```

- for windows
```bash
SET DEBUG=html5-chatroom:* & npm run devstart
```
